# Variables

## Bordes

```scss
$border-radius: 5px;
$border-width: 1px;
$border-style: solid;
$border-color: color(gray6);
```

## Citas

```scss
$blockquote-color: color(gray4);
$blockquote-footer-color: color(gray3);
$blockquote-border-color: color(brand);
```


## Código

```scss
/** Código en línea */
$inline-code-color: color(brand-dark);
$inline-code-background: color(gray7);

/** Código en bloque */
$code-color: color(brand-dark);
$code-background: color(gray2);

/** Entrada de teclado */
$kbd-color: color(gray8);
$kbd-background: color(gray2);
```
